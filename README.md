# Pizza Sales  Report Analysis | SQL & Power BI Dashboard


This project focuses on analyzing pizza sales data using SQL to derive key business insights and then visualizing those insights using Power BI.  The goal is to understand sales performance, customer ordering behavior, and product-level trends to support data-driven decision-making.


## About the Dataset

The dataset contains transactional pizza sales data with the following key attributes:
- Order ID
- Order Date
- Pizza Name
- Pizza Category
- Pizza Size
- Quantity Sold
- Total Price


You can download the dataset [here](https://docs.google.com/spreadsheets/d/1MITlKtNezOi4e80mIU-8b_8nNtNnc5Y3/edit?usp=sharing&ouid=102723999530819284955&rtpof=true&sd=true).
